# Example
Imagine you're checking your inbox for important emails:
::: info Pseudocode
```txt
BEGIN
    WHILE inbox IS NOT empty:
        READ the next email
        IF email IS important:
            MARK email AS unread
            REPLY to the email
        ELSE:
            CONTINUE to the next email
    END WHILE
END
```
:::
